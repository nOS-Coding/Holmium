# Roadmap — Future Features

## Media Playback (Next Major)
Full media control system:
- Spotify integration (web API)
- Local music library playback
- Podcast management
- Volume control, playlists, search
- Tools: `media_play`, `media_pause`, `media_next`, `media_previous`,
  `media_volume`, `media_get_current`, `media_spotify_search`


## Pi Integration
- Raspberry Pi as network Pi-hole + SearXNG
- Pi status, remote commands, display control
- Additional tools: `pi_status`, `pi_service`, `pi_reboot`, `pi_display`

## Smart Home Integration
- Home Assistant API connection
- Lights, thermostat, locks control
- Scene management

## Calendar Integration
- CalDAV sync (Nextcloud, iCloud, Google)
- Event creation, reminders, scheduling
- "What's my day look like?" briefing

## Advanced Image Generation
- LoRA models for consistent characters
- Inpainting/outpainting
- Batch generation with automatic selection

## Multi-User (Future)
- User profiles and preferences
- Session isolation per user
- Shared memory vs personal memory

## Voice Improvements
- Custom wake word training
- Speaker diarization
- Emotion detection in voice

## Mobile Enhancements
- iOS app (Swift)
- Voice shortcuts
- Home screen widgets
- Apple Watch companion

## Performance Optimizations
- Flash attention 2 integration
- Speculative decoding
- Quantized TTS models
- ONNX runtime optimizations
