What is Holmium?

An Arch Linux + OpenRC-based OS that boots directly into a local AI assistant. No browser, no cloud, no telemetry. Your GPU does the work.

  • *Local-first Everything runs on your machine. No data ever leaves your PC. vLLM + llama.cpp for inference.
  • *GPU-accelerated Native CUDA (NVIDIA) or ROCm (AMD) support. Quantized models (Q4_K_M or Q8_0) for maximum performance on consumer hardware.
  • *Dual-boot ready Install alongside Debian, Ubuntu, Fedora, KDE Neon, Nobara, or CachyOS. Auto-detects existing OSes and sets up GRUB chain-loading.
  • *Platform apps Native macOS (AppKit), Windows (WPF), and Android (Material You) companion apps with Chat, Dashboard, and Settings tabs. Linux uses the TUI on tty1.
  • *Private by design Zero telemetry, no accounts required to use the OS. License key is machine-bound (Ed25519-signed) — only used for subscription verification.
  • *Free updates All updates are free while your subscription is active. Rebuild your ISO any time.

Privacy

Holmium collects nothing. The OS has no telemetry, no analytics, no remote logging. The only network call your machine makes is to validate your Ed25519-signed license key on boot (if you have a subscription). License validation contacts our server once per day and transmits only your machine ID — no personal data, no usage statistics, no model prompts. The WireGuard subnet (10.0.0.0/24) is for local device access only; no traffic is routed externally.
